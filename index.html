Git lesson

1. Create new local repository called learning-git-2.
2. Initialize within repository Git.
3. Create in repository index.html file and css directory within style.css file.
4. Do commit with "Init project" title.
5. Add basic structure HTML within file.

1. Dodaj nowy folder o nazwie learning-git-2.
2. Zainicjuj w tym folderze Gita za pomocą komendy "git init".
3. Stwórz w tym folderze podstawową strukturę katalogów oraz dodaj pliki: index.html i css/style.css.
4. Zrób commit z tytułem "Init project" za pomocą komendy "git add ." a następnie "git commit -m "w tytule co dodałeś".
5. Dodaj podstawową strukturę HTML w pliku.
6. Możesz dodać tag za pomocą komendy git tag "w tytule opis tagu". Żeby sprawdzić czy dodany został tag wpisz komendę "git tree".
6a. Za pomocą komendy "git status" możesz sprawdzić czy coś jest do skomitowania.
